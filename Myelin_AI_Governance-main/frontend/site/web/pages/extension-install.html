Install the Myelin Extension

Download the Myelin browser extension package, extract it, then load it unpacked in Chrome or Edge to start using real-time AI governance checks.

  1. Click Download Extension (.zip) below.
  2. Extract the downloaded zip file to a local folder.
  3. Open chrome://extensions in your browser.
  4. Turn on Developer mode.
  5. Click Load unpacked and select the extracted extension folder.
  6. Pin the extension and open the popup to start using it.
