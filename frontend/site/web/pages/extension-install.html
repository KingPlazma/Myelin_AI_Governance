Install the Myelin Extension

The browser extension lives in the repository, not as a downloadable package. Load it unpacked in Chrome or Edge using the local extension folder.

  1. Open chrome://extensions in your browser.
  2. Turn on Developer mode.
  3. Click Load unpacked.
  4. Select the Myelin_AI_Governance/extension folder from this repo.
  5. Pin the extension and open the popup to start using it.
